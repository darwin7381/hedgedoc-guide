## ⚠️ 常見問題

### VitePress 相關

- **Vue 編譯錯誤** → 查看部署指南
- **Build 失敗** → 檢查 markdown 文件中的特殊語法

### Cloudflare Worker 相關

- **主站也壞了** → Worker 判斷邏輯問題
- **樣式不見了** → base path 配置錯誤

### Railway 部署相關

- **部署失敗** → 檢查 Start Command
- **404 錯誤** → 檢查 base path

## 💡 除錯技巧

1. 使用終端機測試：`curl` 命令快速驗證
2. 檢查 Network 面板：確認資源載入
3. 強制重新整理：清除快取
4. 無痕模式測試：避免快取影響
