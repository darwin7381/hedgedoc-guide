從這些歷史文件中，我們學到：

   - 有時候工具就是有限制

   - 不要混在一起
   - 明確標示狀態

3. **驗證很重要**
   - 要對照官方文檔
   - 要實際測試驗證

4. **定期整理文件**
   - 不要讓過時資訊累積
   - 保持文件的清晰性
